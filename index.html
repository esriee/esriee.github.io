Linux I/0 Redirections

When it comes to managing input and output in Linux the kernels handling of I/O redirection is an essential feature that powers many of the system’s functionalities. What is I/O Redirection? I/O redirection in Linux is a mechanism that allows you to change the destination of input and output for commands. by default, linux commands read from the standard input (stdin) and write to the standard output (stdout) and standard error (stderr).

BSPWM

bspwm What is BSPWM? BSPWM is a tiling window manager that arranges windows in a non-overlapping pattern. It is very easy to configure. bspwm doesn’t handle any keyboard or pointer inputs a third party program (e.g. sxhkd) is needed in order to translate keyboard and pointer events to bspc invocations.
