BSPWM

June 5, 2024

What is BSPWM?

BSPWM is a tiling window manager that arranges windows in a non-overlapping pattern. It is very easy to configure. bspwm doesn’t handle any keyboard or pointer inputs a third party program (e.g. sxhkd) is needed in order to translate keyboard and pointer events to bspc invocations.

Follow these step-by-step instructions to install BSPWM on your Linux system:

Install Bspwm

Start by installing BSPWM using your package manager. The exact command may vary based on your distribution. For instance, on Debian-based systems like Ubuntu, you can use

sudo apt install bspwm sxhkd polybar alacritty picom

Arch Linux

sudo pacman -Sy bspwm sxhkd polybar alacritty picom

Void Linux

sudo xbps-install bspwm sxhkd polybar alacritty picom

Clone Dotfiles Repository

Warning: These are my personal configurations. You may need to edit or customize these files to fit your own preferences and setup.

Next, clone your dotfiles repository, which includes BSPWM and other configuration files. Navigate to your home directory and execute

git clone https://github.com/esriee/dots

Move Configuration Files

Once cloned, first go into the cloned repository and locate the BSPWM, Polybar, and other configuration files within the dotfiles repository. Then, move these files to a directory named ~/.config . to move all the configuration files at once, execute this command:

mv ~/dots/* ~/.config/

Running bspwm using xinitrc

To start BSPWM, you need to set it as your default window manager. Create or edit the ~/.xinitrc file to include the following lines

exec bspwm

If you prefer to use a display manager, you might need to select BSPWM on your login screen

That’s it!

The beauty of linux lies in its endless customization possibilities. dive in, experiment :D
